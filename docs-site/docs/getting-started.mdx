# Introduction

MystralNative is a lightweight runtime that lets you run JavaScript/TypeScript games natively using WebGPU. Think of it as "Electron for games" but without Chromium - just your game code, a JavaScript engine, and native WebGPU rendering.

## Why MystralNative?

- **Native Performance**: WebGPU runs directly on the GPU via Dawn or wgpu-native, no browser overhead
- **Familiar APIs**: Use the same WebGPU, Canvas, Audio, and fetch APIs you'd use in a browser
- **Multiple JS Engines**: Choose V8 for full ES2024+ support, QuickJS for small binaries, or JSC on Apple platforms
- **Cross-Platform**: Build for macOS, Windows, Linux, iOS, and Android from one codebase
- **Easy Distribution**: Ship a single binary - no dependencies, no installers

## Quick Example

```javascript
// Your game code - standard Web APIs
const adapter = await navigator.gpu.requestAdapter();
const device = await adapter.requestDevice();
const context = canvas.getContext("webgpu");

// Load assets with fetch
const model = await fetch("file://./assets/model.glb");

// Standard animation loop
function render() {
    // Your rendering code here
    requestAnimationFrame(render);
}
render();
```

## Supported Platforms

| Platform | JS Engines | WebGPU Backend |
|----------|------------|----------------|
| macOS (arm64/x64) | V8, QuickJS, JSC | Dawn, wgpu-native |
| Windows | V8, QuickJS | Dawn, wgpu-native |
| Linux | V8, QuickJS | wgpu-native |
| iOS | JSC | wgpu-native |
| Android | QuickJS | wgpu-native |
